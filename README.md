# GoInspect

An inspect tool like [csgofloat/inspect](https://github.com/csgofloat/inspect) based on [Philipp15b/go-steam](https://github.com/Philipp15b/go-steam).

* [x] Login to Steam
* [x] Login to CSGO
* [x] Could inspect some links
* [ ] More info add to returning data
* [ ] Multi bot account
